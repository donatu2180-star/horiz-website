# horiz-ai.com 画像差し替え — ChatGPT 用コピペセット

## 使い方（3ステップ）

### Step 1. ChatGPT を開く
[chatgpt.com](https://chatgpt.com) を開いて新しいチャットを始める。**画像生成できるモード**を選択（GPT-4o / GPT-5 など）。

### Step 2. 下の「★最初に貼るやつ★」を全部コピーして貼る
ChatGPTが「OK、始めましょう」とか返したら、下の `IMAGE 1` 〜 `IMAGE 17` を1つずつ順番に貼っていく。1つ生成されるごとに **画像を右クリック→保存** → 指定ファイル名にリネーム → `assets/images/` に上書き。

**⚠️ 統一感を保つコツ**:
- **必ず同一チャット内で連続生成する**（チャットを変えると毎回トーンが変わる）
- IMAGE 1 が生成された後、**IMAGE 2 以降を貼る前に「IMAGE 1 と同じ世界観・色調・トーンで続けてください」と一言添える**
- 仕上がりが浮いた画像が出たら「IMAGE 1の雰囲気からズレました。同じ navy / teal / gold / cream 系の静謐なトーンで再生成して」と差し戻す

### Step 3. 全部上書きしたら下記コマンドをターミナルにコピペ
```bash
cd "/Users/ryuki-izumi/Desktop/claude company/horiz-sales/marketing/seo/site" && python3 build.py && python3 -c "
from PIL import Image
import glob, os
for src in glob.glob('assets/images/*.png') + glob.glob('assets/images/*.jpg'):
    base, _ = os.path.splitext(src)
    dst = base + '.webp'
    img = Image.open(src)
    if img.mode in ('RGBA','LA') or (img.mode=='P' and 'transparency' in img.info):
        img.save(dst, 'WEBP', quality=85, method=6)
    else:
        img.convert('RGB').save(dst, 'WEBP', quality=82, method=6)
print('done')
"
```

## 注意点・トラブルシューティング

- **画像のスタイルがバラつく** → 同じChatGPTチャット内で続ければ統一感が出る。新しいチャットで生成すると毎回トーンが変わる
- **解像度が指定より小さい** → ChatGPT は 1024 系が多い。「もう一段階高解像度で」または保存後に Topaz/Upscayl で拡大
- **気に入らない画像** → 「もう少しミニマルに」「teal を控えめに」「明朝の上品さを意識して」「もう少し cinematic に」など微調整指示で再生成
- **PNG が必要なところでJPGになっちゃう** → 「PNGで出力して」と言う。透明背景は不要なので JPG でも実用上OK（リネームすればよい）
